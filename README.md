- The contracts are visable under "List members verbose"

- The credits are transfered at the time of contract creation
- ChatGPT was used to discuss and create code. It worked best when asked to suggest different solutions or reason about solutions. It can not have a lot of code in the "memory" so it often lost track of the bigger picture. It also very easily changed its mind, and if you reminden it about a requirement or to follow MVC it would create a whole new answer. There for, it was not a good source for a final solution, but it was good to give ideas about different ways and ideas to approach or implement something.
